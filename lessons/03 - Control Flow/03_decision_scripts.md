# Decision-Making Scripts

> **Source:** New content
>
> **Notes:** This section applies the week's concepts to a complete, interactive program. Students write a short script that collects user input and makes decisions based on it — e.g., a number guessing game hint, a simple eligibility checker, or a text-based "choose your path" story. The goal is the "it actually does something useful" experience. Provide a worked example and then a prompt for students to extend or create their own.
