# Loop Patterns: Counting, Accumulating, Searching

> **Source:** Ported and expanded from Python Essentials — PE 1.5
>
> **Porting notes:** PE 1.5 shows loop syntax but doesn't explicitly teach loop *patterns*. This section codifies common idioms as named patterns so students build a mental toolkit: (1) **counter pattern** — increment a variable each iteration; (2) **accumulator pattern** — build up a total or a list; (3) **search pattern** — loop until a condition is met, then break. Also cover nested loops with a simple example (e.g., multiplication table). Explicitly compare `for` vs. `while` — when each is the better choice.
