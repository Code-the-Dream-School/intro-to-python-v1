# Sorting and Searching Algorithms

> **Source:** New content — not in Python Essentials
>
> **Notes:** This is the algorithmic thinking content unique to this course. Keep it conceptual and grounded — the point is not to memorize algorithms but to practice decomposing a problem into steps and implementing that in code. Cover:
> - **Linear search:** iterate through a list looking for a match; return the index or `None`
> - **Binary search:** conceptual explanation (requires sorted list; halving the search space); implement iteratively with a `while` loop
> - **Bubble sort or selection sort:** pick one; walk through the steps; implement with nested loops
> - Big-picture takeaway: different algorithms have different performance characteristics; Python's built-in `sort()` is better for production use, but understanding how sorting works matters
