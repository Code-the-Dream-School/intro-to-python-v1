# Parsing Lines Into Data Structures (Lists of Dicts)

> **Source:** Ported and expanded from Python Essentials — PE 2.2 (File Handling)
>
> **Porting notes:** PE 2.2 shows CSV file handling but doesn't explicitly teach the "read a CSV into a list of dicts" pattern as a first-class concept. This section makes that pattern explicit and central. Cover: using `csv.DictReader` to iterate and collect rows into a list; accessing fields by header name; the resulting data structure (list of dicts); then looping over and filtering/transforming that structure using skills from Weeks 4–6. The goal is to show students a complete data pipeline in miniature: file → structured Python data → useful output.
