# pip and Virtual Environments (Conceptual)

> **Source:** Ported from Python Essentials — PE 2.6 (Virtual Environments) and PE 1.1 (pip section)
>
> **Porting notes:** Students activated a virtual environment in Week 2 but didn't fully understand why. Now revisit it with context. Take PE 2.6 content and consolidate with the pip coverage from PE 1.1. Cover: what `pip` is and how PyPI works; installing a package (`pip install requests`); what a virtual environment is and why project isolation matters; creating/activating `.venv`; `pip freeze > requirements.txt` and `pip install -r requirements.txt`. This section is largely conceptual — students have already done the steps, now they understand why.
