# Git Setup & Configuration

> **Source:** New content (adapted from Intro to Programming — JavaScript)
>
> **Notes:** Students created a GitHub account and repo conceptually in Week 1. Now they install Git locally and configure it. Cover: downloading and installing Git; `git config --global user.name` and `user.email`; verifying with `git --version`; the conceptual model of commits (snapshots of your project over time); the difference between Git (local version control software) and GitHub (cloud hosting for Git repos). Students do NOT make their first commit this week — that happens in Week 3 alongside Control Flow.
