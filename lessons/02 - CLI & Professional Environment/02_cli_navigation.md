# CLI Navigation

> **Source:** New content (adapted from Intro to Programming — JavaScript)
>
> **Notes:** This is the first time many students have used a terminal. Cover: what the terminal/shell is and why developers use it; opening a terminal (Mac: Terminal or iTerm, Windows: Git Bash); the prompt; essential commands: `pwd`, `ls` (Mac/Linux) / `dir` (Windows), `cd`, `mkdir`, `touch`/`echo`; absolute vs. relative paths; tab completion. Keep examples concrete — navigate to a folder and create a file. Reassure students that memorizing commands comes with practice.
