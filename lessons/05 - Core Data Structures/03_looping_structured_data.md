# Looping Over Structured Data

> **Source:** Ported and expanded from Python Essentials — PE 2.1 (iteration examples throughout)
>
> **Porting notes:** PE 2.1 shows iteration examples inline with each data structure. This section consolidates and expands that into a standalone treatment. Cover: iterating over lists with `for`; iterating over dict keys, values, and `.items()`; iterating over sets (unordered — show why order isn't guaranteed); **lists of dictionaries** as a pattern for tabular data (e.g., a list of student records). The list-of-dicts pattern is important to establish here because it recurs in file handling (Week 7) and API responses (Week 9).
